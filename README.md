## SimFeedback + Servo 80AST Pins

### Arduino Pins

| Pin   | Motor 1 | Motor 2 | Motor 3 | Motor 4 | DB25 PIN |
|-------|---------|---------|---------|---------|----------|
| STEP  | 8       | 9       | 10      | 11      | 3        |
| DIR   | 4       | 5       | 6       | 7       | 4        |
| ENA   | A0      | A1      | A2      | A3      | 6        |
| 5V    | 5V      | 5V      | 5V      | 5V      | 9        |
| GND   | GND     | GND     | GND     | GND     | 5, 10, 14 |

### DB25 Controller Cable Mapping

| DB25 Pin | Label |
|----------|-------|
| 3        | **P+**    |
| 4        | **D+**    |
| 6        | **SON**   |
| 9        | **24V**   |
| 5,10,14  | **P- / D- / 0V** |



## Parameters

` P-006 ; 25 `<br>
` P-007 ; 30 `<br>
` P-019 ; 60 `<br>
` P-021 ; 20 `<br>
` P-029 ; 16 = 100mm, 32 = 200mm stroke limit `<br>
` P-040 ; 30 `<br>
` P-098 ; 1 (Force Enable) (비상버튼 사용시; 0)`<br>

## 3D Printing setup

bump stop / slider 0.4mm 노즐 출력 (정밀성)

0.4mm 노즐 출력 세팅
| 설정 항목           | 권장 값                   |  
| --------------- | ---------------------- | 
| 노즐 직경           | **0.4mm**              |   
| 레이어 높이          | **0.2mm**              |   
| Perimeter(외벽 수) | **5개**                 |  
| 상/하단 레이어        | 8개  |  
| 인필 밀도           | **25%**, Grid 패턴       |  
| 브림              | **5mm**    |
